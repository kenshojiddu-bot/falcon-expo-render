# 沙龙海报视觉差异化设计

## 目标

在保持四张海报文件名和线上网址不变的前提下，强化每张海报的主题识别度，并解决海钓缺少船体、主报名海报与猎鹰主题背景重复、无二维码海报版式偏左的问题。

## 视觉方案

- 主报名海报：改用猎鹰展现场照片作为全幅背景，保留报名二维码、活动概述、时间地点和四节点流程。使用铜金与白色，呈现展会与商务氛围。
- 茅台主题海报：保留无品牌酒液特写，使用中国红半透明色调与金色文字，标题、说明和底部信息全部水平居中。
- 海钓主题海报：替换为可明确看见海钓船和海面的出海画面，使用深海蓝色调、冰蓝与白色文字，全部水平居中。
- 出海主题海报：保留猎鹰特写，使用沙特绿与金色文字，全部水平居中，与主报名海报形成明显的图像差异。

## 排版规则

- 三张无二维码海报共用居中版式：顶部系列名、两行主口号、主题说明、底部日期和活动名均以画布中线对齐。
- 主口号保持最大字号，主题说明次之，底部信息收束画面。
- 三张配套海报不绘制二维码；二维码仅保留在主报名海报右下角。

## 验收

- 四张图片保持 RGB PNG、1080×1920。
- 主报名海报二维码解码为 `https://falcon-expo.onrender.com/salon.html`。
- 三张配套海报不检测到二维码。
- 海钓海报能在 390×693 手机预览中辨认出船体和海面。
- 主报名海报与出海主题海报使用不同背景图片。
